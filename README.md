# Getting Started

## Prerequisites

### Installing dependencies
- pip
    ```bash
      cd ./app/entirety
      pip install -r requirements.txt
    ```
- pre-commit
    ```bash
      pre-commit install
    ```

## Usage

Starting the Django server:

```bash
  python manage.py runserver
```
